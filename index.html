Pocket Zoo

Created a mobile gacha game in which players collect virtual animals. The app uses real-time location services to modify the draw pool, making certain animals exclusive to specific real-world locations. The server is deployed on a Microsoft Azure VM for persistent uptime. The game is designed for the Google Pixel 3 on API 29+.

Easy Check-Inn

Collaborating with three classmates, I worked on an automated hotel check-in service written in Python. I created a custom dataset from a subset of a public dataset and applied transfer learning to train a face mask detection model. The model achieved a 99.2% accuracy with under 0.5s runtime.

Categorical Slice Plots

Facebook's Ax library natively supports slice plots, but only for numerical values. I created a patch to put over the Ax slice plot function to add support for categorical values as well. I then used React and HighCharts to integrate these categorical slice plots into a page.

CronJob Notification Service

I implemented an email notification service using CronTab/CronJobs. Clients receive notifications for progress reports at selected intervals as well as warnings and alerts. Warning notifications may be disabled but alerts will always be sent.

Music Recommendation App

Collaborating with three classmates, I worked on a music recommendation app written in Python. We modified a VGG-19 model and trained on a custom dataset of 4376 h5 files to create a music genre classifier that classifies based on spectrograms. The app would then provide a song recommendation based on the genre. We created the GUI for the app using the Tkinter Python library.

Pose Recognition Module

A Python script that uses YOLOv5 to perform real-time pose recognition. The module was trained on images from a custom dataset derived from the COCO dataset for two possible predictions: sitting, and standing. The script was used in the team's entry for the Robocup@Home Education competition in 2020 and we placed 2nd internationally.
